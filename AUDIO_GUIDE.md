# 音频功能使用指南 (Audio Guide)

## 1. 功能概述
本项目内置了韩语单词发音功能，支持手动播放和自动朗读。所有音频资源经过深度压缩优化，确保小程序加载速度和稳定性。

## 2. 使用方法

### 手动播放
- 在练习界面的右上角控制栏中，点击 **喇叭图标** 即可播放当前单词的发音。
- 如果当前单词没有对应的音频资源，喇叭图标会显示为半透明状态。

### 自动朗读
- 点击控制栏左侧的 **设置图标**（齿轮形状）。
- 在弹出的菜单中选择 **“开启自动朗读”**。
- 开启后，每次切换到新的单词时，系统会自动播放发音。
- 如需关闭，再次点击设置图标并选择 **“关闭自动朗读”** 即可。

## 3. 音频资源说明

### 资源优化
- **格式**: MP3 (CBR, 48kbps, Mono, 22050Hz)
- **大小**: 当前 `beginnerWords` 音频包大小约 **1.2MB**。
- **合规性**: 微信小程序主包限制为 2MB，优化后的音频文件完全符合要求，不会影响发布。

### 技术实现
- 使用 **Audio Sprite**（音频雪碧图）技术，将所有单词音频合并为一个文件，减少网络请求和文件句柄开销。
- 采用 **重编码拼接** 策略，确保时间轴精准，解决了直接拼接可能导致的“截取不全”或“偏移”问题。

## 4. 常见问题

**Q: 为什么有的单词没有声音？**
A: 请检查该单词是否在录音列表中。如果喇叭图标变暗，说明该单词暂无音频。

**Q: 自动朗读有时候不响？**
A: 请确保手机未处于静音模式。iOS 用户请注意侧边静音键。此外，网络加载（首次）可能需要一点时间。

**Q: 如何添加新的音频？**
A: 请将新的 MP3 文件放入 `data/voice/新目录` 中，然后运行 `node tools/process_audio.js` 生成新的雪碧图和索引。
